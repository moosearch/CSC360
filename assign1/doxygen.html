Doxygen

Overview

Doxygen is a piece of software that generates a HTML website documentation and API for your code base solely from your source code. (This is very similar to javadoc for those who may have worked with that). Doxygen allows people to review your code in HTML and clusters together relevant pieces automatically. All that is required from the programmer's point-of-view is to include minimal markup for doxygen in your source code to be used during its processing. When using the markdown notation, for example, the formatting of documentation is very simple and leads to pleasant results. In addition, it is easy and natural to read in the actual source file. To demonstrate, this site was actually made using markdown -- see the source here.

Install

Download Doxygen from (A binary for Windows can be found if you scroll down):

  • http://www.stack.nl/~dimitri/doxygen/download.html

Install Doxygen using the default settings.

Test

  • Run doxywizard. It should be in the start menu or can be found in C:\Program Files\doxygen\bin\doxywizard.exe.
  • Choose File->Open and select the provided default "Doxyfile" configuration file, which should be at the same level as your "code" and "html" directories.
  • Click on the Run tab. Click Run doxygen.
  • When it has completed, click Show HTML Output.

If everything worked correctly, your web browser should open with a nice doxygen generated website.

Basic Usage

  • For formatting, I recommend using Markdown notation. It is very straightforward and readable.
  • Change code/mainpage.md to edit your index page.

To add comments to doxygen:

 Adding two new lines will create a new paragraph.
